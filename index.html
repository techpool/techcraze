    MANUAL ROBOTICS

    Manual robotics is where the participant needs to control his/her own bot via a remote. The bot can't go on it's own. This is an event where the participant not only needs to make a great design but also needs to have great proficiency in the control mechanism of the bot so as to pass all the hurldes and take away the prize

    We have a wide range of events for MANUAL ROBOTICS, ranging from a bot which has to pass the real world terrains to a bot where it has to play carrom. We even have an event where the bot needs to stay afloat over water and move the obstacles to proper positions, so for this event you need to be extra careful about the insulation.
